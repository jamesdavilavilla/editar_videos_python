# Reddit Faker Bot 
Este bot genera 3 videos diarios estilo Reddit automáticamente.

## Requisitos
- Python 3 instalado
- Ejecuta: pip install -r requirements.txt

## Uso
1. Coloca un video vertical sin copyright en la carpeta "gameplay" llamado "minecraft_sample.mp4"
2. Ejecuta el bot con:
    python main.py
3. Los videos generados estarán en la carpeta "output"
